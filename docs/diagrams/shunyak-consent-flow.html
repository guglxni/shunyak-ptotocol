Shunyak Consent Registration Flow

Identity-backed consent registration and on-chain anchoring

Identity Phase

  • - DigiLocker request is created or resumed.
  • - Backend waits for authenticated status.
  • - Claim context is resolved before anchoring.

Validation Phase

  • - AlgoPlonk payload shape is validated.
  • - App-call payload is assembled by API layer.
  • - Registrar path writes consent state on-chain.

Output Phase

  • - Frontend receives tx metadata and consent token.
  • - Consent state becomes queryable by execution tools.
  • - Explorer-verifiable anchor is available immediately.
