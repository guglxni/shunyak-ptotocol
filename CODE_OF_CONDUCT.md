# Code of Conduct

## Our Commitment

This project aims to provide a respectful, inclusive, and constructive environment for everyone.

## Expected Behavior

- Be respectful and professional.
- Assume good intent and discuss ideas, not people.
- Give actionable, evidence-based feedback.
- Help keep discussions technical and collaborative.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, abusive language, or intimidation.
- Publishing private information without consent.
- Any conduct that creates an unsafe or hostile environment.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and project-related community spaces.

## Enforcement

Maintainers may remove comments, close threads, or restrict participation for conduct violations.

## Reporting

If you experience or witness unacceptable behavior, report it to maintainers via repository maintainers/channels.

## Attribution

This policy is inspired by common open-source community standards and adapted for this repository.
